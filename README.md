# Android Studio Installation and Running Applications

This repository contains the experiment/documentation for installing and running applications using Android Studio.

## Aim
To install Android Studio and configure the environment for Android application development.

---

## Software Requirements
- Java JDK 5 or later
- Java Runtime Environment (JRE)
- Android Studio
- Android SDK
- Android Virtual Device (AVD)

---

## Step 1 – System Requirements
The required tools to develop Android applications are open source and can be downloaded from the web.

### Required Software
- Java JDK
- JRE
- Android Studio

---

## Step 2 – Setup Android Studio
Android Studio is the official IDE for Android application development and works based on IntelliJ IDEA.

### Installation Procedure
1. Download Android Studio from the official website.
2. Run the installer.
3. Make sure Java JDK is already installed.
4. Provide the JDK path during installation.
5. Select required components:
   - Android Studio
   - Android SDK
   - Android Virtual Device
   - Performance tools
6. Choose installation location.
7. Allocate RAM for the Android Emulator.
8. Complete SDK package extraction and installation.
9. Launch Android Studio.

---

## Creating a New Project
1. Open Android Studio.
2. Select **Start a New Android Studio Project**.
3. Enter:
   - Application Name
   - Package Name
   - Project Location
4. Select Minimum SDK version.
5. Choose an Activity template.
6. Finish setup to open the development environment.

---

## Step 3 – Create Android Virtual Device (AVD)
To test Android applications, create a virtual Android device.

### Procedure
1. Open **AVD Manager**.
2. Click **Create New Virtual Device**.
3. Select device configuration.
4. Download/select Android system image.
5. Finish AVD setup.

If the AVD is created successfully, the environment is ready for Android application development.

---

## Result
Android Studio was installed successfully and the Android Virtual Device was configured for running Android applications.

## Conclusion
The Android development environment was successfully set up using Android Studio, SDK, and AVD tools.
